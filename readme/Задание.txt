Открыть папку "Материалы" и памятку по HTML
--------------------------------------------------

1. Создать папку "task1" в папке "Проэкты/lessons/HTML/"
2. Создать новый html файл под именем "index.html" и внутри него написать базовую структуку html-документа.
3. Написать заголовок страницы который отобразиться во вкладке страницы в браузере (тег title)
4. Написать тег "div" и в нем разместить текст из файла "text.txt"
5. Выделить тегами заголовок первого уровня
6. Разбить вставленный текст на 3 абзаца соответствующими тегами, перед началом каждого абзаца выделить тегами заголовки второго уровня.

7. Выделить произвольные участки текста:
- жирным, 
- курсивом, 
- в нижнем (найти формулу воды) и верхнем индексе (найти метры в квадрате), 
- Подобрать участок текста и расставить теги маркированного и нумерованного списков, 
- сделать перенос строки в любом месте в тексте.

8. Копировать и вставить картинку из "Материалы/image.jpg" после последнего абзаца и дать ей описание в соответствующем атрибуте.

9. Переписать код таблицы, ниже, в свой файл в любое место:
<table border="1">
	<tr>
		<td>1</td>
		<td>2</td>
	</tr>
	<tr>
		<td>3</td>
		<td>4</td>
	</tr>
</table>


10. Создать еще один html файл под именем "index.htm", внутри которого будет текст из файла "text.txt" вообще без каких-либо тегов.

11. В файле "index.html" вствить ссылку на файл "index.htm"

12. Проверить весь код в валидаторе

13. В любом месте в пределах body файла "index.html" написать комментарий, в котором отметить сколько времени заняло выполнение задания.

14. Пришли мне ссылку на свое выполненное задание, через яндекс.диск
